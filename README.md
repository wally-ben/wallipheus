This is a Slack App made by Wally for their personal channel: #the-wally-cave
it includes 5 commands: 
- /wallipheus-ping: returns the latency
- /wallipheus-greet: sends a greeting! currently it's just fetched from an array.
- /wallipheus-cat: fetches a Cat Fact from a public api and sends it to the channel.
- /wallipheus-remind: reminds you after x amount of minutes of a specific message that you input.
- /wallipheus-daily: The Slack app has a build in daily reminder at 9am, this command lets me (Wally) cange that reainder's message.

Wallipheus also welcomes new members autmatically.
